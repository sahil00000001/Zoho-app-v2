Engineering

Built for speed & reliability

⚡
API Cache
60s in-memory cache. Stable data (departments, leave types) cached 5 min. Zero re-fetches on navigation.
🔗
In-flight Dedup
If two components request the same URL at once, only one HTTP call fires. Both share the result.
🔥
Keepalive
Pings backend every 4 min. Prevents Vercel serverless cold starts (3–8s delays) during business hours.
🗄️
DB Indexes
12 targeted composite indexes. User.managerId, Leave.(status+dates), AuditLog.action — 5–20× query speed.
🔄
N+1 Fixed
Leave balance was N queries (1 per leave type). Now 2 parallel queries + in-memory grouping.
📦
Parallel Auth
getMe + getPermissions now run in parallel on login. Startup time cut by ~40%.
💀
allSettled
Independent API calls use Promise.allSettled — one failure never blanks the whole page.
🦴
Skeleton UI
Shimmer skeletons during load — never a blank white screen. Page feels instant.
📱
Mobile-First
Hamburger sidebar drawer on mobile. Responsive grids, wrapping headers. Full UX on any screen size.
🔄
Live Role Sync
Auth middleware fetches current role from DB every request. Role promotions take effect instantly — no logout required.
